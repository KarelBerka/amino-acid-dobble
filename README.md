# 🧬 AA-Dobble (Biogenní aminokyseliny)

Výukový generátor a trenažér karetní hry Dobble zaměřený na 21 biogenních aminokyselin.

🌐 **Živá aplikace (GitHub Pages):** [https://karelberka.github.io/amino-acid-dobble](https://karelberka.github.io/amino-acid-dobble)  
🌐 **BioDobble Portal:** [https://karelberka.github.io/bio-dobble](https://karelberka.github.io/bio-dobble)

---

## 🎮 Parametry balíčku

- **Řád Dobble:** $q = 4$
- **Počet karet:** 21
- **Počet aminokyselin:** 21 (20 standardních + selenocystein)
- **Počet symbolů na kartě:** 5

---

## 🎨 Reprezentace na kartách

- České, anglické, německé a francouzské názvy
- 3-písmenné a 1-písmenné zkratky
- 2D strukturní vzorce
- 3D ball-and-stick PyMOL modely
